# Travel Guide Website

## Project Description
A responsive travel guide website with multiple pages and interactive features.

## Features
- Responsive design
- API integration for destinations
- Form validation
- Accessible markup

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- REST Countries API

## Installation
1. Clone this repository
2. Open `index.html` in your browser
